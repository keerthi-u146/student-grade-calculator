# student-grade-calculator
A simple Java console program that calculates a student's total marks, percentage, and grade, based on input for three subjects.

Technologies Used :
1.Java.
2.Visual Studio Code (or any Java IDE).

Features :
1.Input student name and marks for 3 subjects.
2.Calculates total marks and percentage.
3.Assigns grade based on percentage.
4.Handles invalid input (non-numeric entries).
5.Displays results clearly in the console.

Project Description: 
-Built a Java application to calculate student marks
-Accepted marks for multiple subjects as input
-Calculated total and average marks
-Assigned grade based on performance using conditions
-Displayed final result in console output

How to Run :
1.Download or clone the repository.
2.Open terminal in the project folder.
3.Compile the program : javac StudentGradeCalculator.java.
4.Run the program : java StudentGradeCalculator.
5.Enter the student name and marks for three subjects.
6.The program displays total marks, percentage, and grade.
